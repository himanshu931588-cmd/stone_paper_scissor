# stone_paper_scissor
Make a stone paper scissor game using python (functions,looping,conditional statement)
🪨 📄 ✂️ Rock Paper Scissors: Python Edition
A lightweight, terminal-based implementation of the classic hand game. This project is designed for simplicity, featuring a "Human vs. Computer" logic flow with instant result validation.
✨ Features
>Interactive Gameplay: Simple text-based input for the user.

>AI Opponent: Randomly generated moves using Python's random module.

>Score Tracking: (Optional feature) Keeps track of wins, losses, and draws during a session.

>Input Validation: Handles typos or invalid entries gracefully to ensure the game doesn't crash.
